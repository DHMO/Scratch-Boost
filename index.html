    About

    Scratch Boost is an extension for the website Scratch that adds several new blocks.

    Installation

    To install Scratch Boost all you have to do is drag this bookmarklet Scratch Boost. Then, when viewing a Scratch project click the bookmarklet to install Scratch Boost.
